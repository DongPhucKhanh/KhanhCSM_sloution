### 🚀 Buổi 3: Bảo mật hệ thống & Quản lý nội dung đa phương tiện
* **Nhiệm vụ:** Phân quyền hệ thống và tích hợp Editor.
* **Chi tiết công việc:**
  * Áp dụng thuật toán băm (Hashing) bảo mật mật khẩu một chiều cho thực thể `User`.
  * Triển khai thuộc tính `[Authorize(Roles = "Admin")]` để khóa các Controller quản trị, chặn truy cập trái phép.
  * Xây dựng luồng Đăng nhập / Đăng xuất bằng Session/Cookie cho quản trị viên.
  * Tích hợp thành công trình soạn thảo Rich Text (CKEditor) vào trang viết bài, hỗ trợ định dạng HTML và Upload ảnh vật lý.
