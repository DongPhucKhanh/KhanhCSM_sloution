### 🚀 Buổi 8: Kiểm thử hệ thống, Xử lý lỗi (Bug fixing) & Báo cáo
* **Nhiệm vụ:** Đánh giá chất lượng và hoàn thiện hồ sơ.
* **Chi tiết công việc:**
  * Code logic kiểm tra số lượng tồn kho (`StockQuantity`), chặn bắt lỗi hiển thị cảnh báo đỏ khi khách mua vượt giới hạn.
  * Áp dụng `dangerouslySetInnerHTML` để render bài viết HTML kỹ thuật xe an toàn.
  * Vận hành End-to-End, theo dõi F12 Tab Console đảm bảo sạch lỗi (Zero Errors/Warnings).
  * Đóng gói Source code và hoàn thiện tài liệu Báo cáo Đồ án.
